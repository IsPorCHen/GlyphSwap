# GlyphSwap

[EN] An automated tool for intercepting, OCR-processing, and translating text on images in your browser.
[RU] Инструмент для автоматического перехвата, OCR и перевода текста на изображениях в браузере.

## Installation / Установка
1. Install [Tampermonkey](https://tampermonkey.net/).
2. Click on the link to the [latest release](#).
3. Confirm installation.

## Language Switching / Смена языка
Use the hover widget in the top-right corner of the image to switch between EN and RU.
