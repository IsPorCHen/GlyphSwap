# GlyphSwap

[English](README.md) | [Русский](README.ru.md)

## О проекте
**GlyphSwap** — это пользовательский скрипт (Userscript) для браузера, который позволяет переводить текст прямо на изображениях. Больше не нужно копировать текст или использовать сторонние переводчики — просто наведите курсор на картинку, и GlyphSwap распознает текст и наложит перевод поверх оригинала.

## Основные возможности
* **Удобный виджет:** Кнопка перевода появляется только при наведении курсора на изображение.
* **Мощное распознавание:** Использует движок Yandex OCR для качественного извлечения текста.
* **Пакетный перевод:** Переводит все найденные текстовые блоки одновременно, сохраняя контекст.
* **Безопасность:** Не меняет оригиналы изображений на странице, работая через наложение (overlay).

## Установка
1. **Менеджер скриптов:** Убедитесь, что у вас установлен [Tampermonkey](https://www.tampermonkey.net/) или [Violentmonkey](https://violentmonkey.github.io/).
2. **Установка:** Нажмите на ссылку ниже, чтобы установить последнюю версию скрипта:
   [**Установить GlyphSwap**](https://github.com/IsPorCHen/GlyphSwap/raw/main/dist/glyphswap.user.js)

## Как пользоваться
1. После установки перейдите на любую страницу с картинками.
2. **Наведите курсор** на изображение. В правом верхнем углу появится кнопка **"A文 Swap"**.
3. Нажмите на неё.
4. Скрипт автоматически обработает картинку, распознает текст и выведет перевод.
